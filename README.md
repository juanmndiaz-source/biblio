# 📚 Biblioteca Digital CEICH

Plataforma colaborativa de la **Facultad de Ingeniería y Ciencias Hídricas (FICH)** para centralizar y compartir materiales de estudio entre estudiantes.

## ✨ Características

- 🎓 **Base de datos centralizada** de apuntes y materiales
- 🤝 **Plataforma colaborativa** entre estudiantes  
- 📱 **Diseño responsive** (mobile-first)
- 🎨 **Interfaz moderna** y accesible
- ⚡ **Búsqueda rápida** de contenidos
- 🏗️ **Arquitectura escalable** y mantenible
- 🔐 **Preparada para backend** y autenticación

## 🏗️ Estructura del Proyecto

```
testttt/
├── css/              # 🎨 Estilos CSS
├── js/               # ⚙️ Scripts JavaScript
├── img/              # 🖼️ Imágenes y recursos
├── data/             # 📊 Datos en JSON
├── pages/            # 📄 Páginas HTML
├── docs/             # 📚 Documentación
└── index.html        # 🏠 Página principal
```

**Para más detalles:** Ver `/docs/ESTRUCTURA.md`

## 🚀 Cómo Empezar

1. **Clonar el repositorio:**
   ```bash
   git clone https://github.com/ceich/biblioteca-digital.git
   cd testttt
   ```

2. **Abrir en el navegador:**
   - Simplemente abre `index.html` en tu navegador
   - O usa Live Server en VS Code

3. **Personalizar:**
   - Edita el archivo `css/variables.css` para cambiar colores
   - Agrega tus imágenes en la carpeta `img/`
   - Actualiza el contenido en `/pages/`
